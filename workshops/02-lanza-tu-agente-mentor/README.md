# Lanza tu propio Agente Mentor

**Promesa:** Empieza a construir con una IA que te explica mientras construye lo que le pides.

## Objetivo

Que el asistente entienda la diferencia entre usar un LLM y operar un harness, y salga con un Pi Coding Agent local listo para ayudarle a construir.

## Audiencia

Builders, estudiantes, operadores técnicos y personas que quieren aprender construyendo con un agente local.

## Resultados esperados

- Distinguir LLMs vs harnesses.
- Entender los fundamentos operativos de Pi Coding Agent.
- Instalar o preparar un agente local.
- Usar el agente para explicar, planear y ejecutar un cambio pequeño en un proyecto.

## Contenido base

### LLMs vs Harnesses

- Un LLM responde; un harness opera con contexto, herramientas, memoria/proyecto y límites.
- El harness define qué puede leer, escribir, ejecutar y revisar el agente.
- Un buen agente mentor explica decisiones, propone pasos, ejecuta cambios pequeños y verifica.

### Pi Coding Agent fundamentals

- Instalación/configuración base.
- Modelo/proveedor y credenciales.
- Proyecto local, permisos, comandos y ciclo de trabajo.
- Prompts de trabajo: pedir explicación, plan, implementación y verificación.

### Deploy your local Pi Coding Agent

- Preparar entorno.
- Ejecutar agente en proyecto de ejemplo.
- Pedirle que genere una mini app, endpoint o README.
- Revisar output y confirmar que realmente corrió o produjo archivo.

## Artefacto del workshop

Un Pi Coding Agent local configurado y una primera tarea completada con explicación paso a paso.
